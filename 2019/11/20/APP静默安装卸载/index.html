APP静默安装卸载

发表于 2019-11-20 | 分类于 Android ， 技术积累 |
字数统计: 840 | 阅读时长 ≈ 4

前提条件

  • root 过的 Android 设备
  • 一台能运行 Android 代码的电脑

测试结果

  • Android 5.1 的 root 过的定制设备实现静默安装卸载
  • Android 8.0.0 的 root 过的华为荣耀实现静默安装卸载

实现过程

  • AndroidManifest.xml 中添加权限
  • 检查是否有 root 权限
  • 静默安装
  • 静默卸载

总结

  • 若要实现静默安装卸载，应用需获得 root 权限；

  • 若要实现静默安装，需获取到目标 apk 的路径；

  • 若要实现静默卸载，需获取到目标应用的包名；

  • 静默安装卸载均采用

    • 静默卸载：LD_LIBRARY_PATH=/vendor/lib:/system/lib pm uninstall -k 目标应用的包名
